# Sentiment-Analysis

The project Sentiment Analysis of Tweets for US Airline with Interactive Dashboard using 
ML Model was developed to train an ML Model about sentiments, then to classify the 
sentiment using the trained ML model, and Customers or Users can interact with the 
system using the dashboard created with python and Streamlit Framework to manipulate 
the classification results.

<h1>Using the App</h1>
<p>These instructions will get you a copy of the project up and running on your local 
machine for development and testing purposes.</p>

<h2>Prerequisites</h2>
<code>python== 3.0 or higher</code>

<h2>Installing</h2>
<pre>open terminal and type</pre>
<code>git clone https://github.com/benjaminagbo/Sentiment-Analysis.git</code><br><br>

<h4>or simply download using the url below</h4>
<code>https://github.com/benjaminagbo/Sentiment-Analysis.git</code><br>

<h2> To run the program in local server use the following command </h2>
<code>streamlit run Dashboard.py</code>

<p>Then go to http://127.0.0.1:8000 in your browser</p>
